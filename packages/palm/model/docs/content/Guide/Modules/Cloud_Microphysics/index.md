# Cloud Microphysics Models

## Introduction

PALM allows to simulate clouds, precipitation, and all associated microphysical processes. There are two fundamentally different approaches for this: a classical bulk parameterization and novel Lagrangian particle-based microphysics.

The bulk microphyiscs model should always be used if no special interest is given in the exact microphysical processes. However, even with this highly parameterized schemes clouds and their development can be simulated with a high level of detail when using a complete mixed-phase two-moment microphysics scheme (based on the implementation of Seifert and Beheng, 2006 in the ICON model).

The Lagrangian particle-based microphysics is a highly sophisticated method to simulate cloud microphysics based on first principles. A detailed description can be found via the links given below.
